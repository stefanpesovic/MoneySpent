# Novčanik

Lični i restoranski budžet kao PWA (Progressive Web App) — gotovina i kartica, u EUR.
Sve u jednom fajlu (`index.html`), bez build koraka i bez eksternih biblioteka.
Radi 100% oflajn i može da se instalira na iPhone preko Safari-ja (*Dodaj na početni ekran*).

## Funkcije
- **Restoran** — dnevni pazar (keš/kartica), nedeljni i mesečni pregled.
- **Lično** — prihodi i rashodi, dva novčanika (Račun, Novčanik), podizanje keša.
- **Pregled** — ručno crtani grafici (14 dana, keš vs kartica, troškovi po kategoriji).
- **PIN zaključavanje**, izvoz/uvoz rezerve (.json) i izvoz pazara (CSV).

## Pokretanje
Otvori `index.html` u browseru. Nema instalacije ni zavisnosti.

## Hosting
Statički sajt — može na Netlify, GitHub Pages, ili bilo koji statički host.
